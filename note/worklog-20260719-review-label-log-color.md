# Review Agent 명칭 및 로그 색상 통일

## 요청

- `risk-review-agent`의 사용자 표시 명칭을 `risk-review-agent(초안 검토 및 보완)`으로 변경한다.
- Agent 로그 블록의 배경색과 테두리색을 로그 종류와 관계없이 동일하게 표시한다.

## 변경 원칙

- 로그 종류를 나타내는 `Agent`, `Skill`, `Tool`, `검증`, `주의` 글자 표시는 유지한다.
- 오류 여부도 색상 대신 종류 글자와 제목으로 확인할 수 있게 한다.
- 부모 로그와 내부 자식 로그 모두 흰색 배경과 동일한 회색 테두리를 사용한다.

## 성공 기준

- 화면 어디에서도 이전 `위험도 계산·근거 검토` 명칭이 표시되지 않는다.
- 모든 로그 카드가 동일한 배경색과 테두리색을 사용한다.
- Frontend typecheck와 build가 통과한다.
