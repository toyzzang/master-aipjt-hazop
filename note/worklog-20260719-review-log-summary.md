# Review Agent 실시간 로그 요약

## 요청

- 초안 검토 및 보완 내역을 실시간 Agent 로그에서 한 건씩 모두 표시하지 않는다.
- 총 보완 건수와 담당자 확인 필요 건수만 표시한다.
- 상세 검토 내역은 완료 후 결과 표에 유지한다.

## 변경

- `_record_review_findings`의 Row별 반복 이벤트를 요약 이벤트 1건으로 변경한다.
- 요약 형식: `총 N건을 보완했습니다. 담당자 확인이 필요한 항목은 M건입니다.`
- 상세 `message`, `resolution`, `category`는 `HazopResult.review_findings`에 그대로 보존한다.

## 성공 기준

- Review Agent 블록에 검토 요약 로그가 정확히 1개 표시된다.
- 실시간 로그에 개별 검토 발견·반영 문장이 표시되지 않는다.
- 완료 후 `초안 검토 및 보완 내역` 표에는 상세 기록이 표시된다.
