# 사고이력 분석 절차

1. 동일 Node, 동일 변수, 동일 Guideword 이력을 우선 찾는다.
2. 동일 물질 또는 유사 물질 사고이력을 찾는다.
3. 동일 설비 계통의 알람, Near Miss, 정비 이력을 찾는다.
4. 최근성과 반복성을 빈도 후보에 반영한다.
5. 실제 인명·설비·환경 피해 범위를 강도 후보 근거에 반영한다.
6. 데이터가 없으면 `사고이력 데이터 확인 필요`를 근거에 남긴다.
