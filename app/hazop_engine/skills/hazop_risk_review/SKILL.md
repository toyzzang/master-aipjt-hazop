# HAZOP 위험성평가 검토 절차

검토 기준:

1. 입력 Excel에 없는 Node, 변수, Guideword를 만들지 않았는지 확인한다.
2. 빈도가 1~5 범위인지 확인한다.
3. 강도가 1~4 범위인지 확인한다.
4. 판단근거, 빈도근거, 강도근거가 비어 있지 않은지 확인한다.
5. 고위험 물질 누출인데 강도가 너무 낮지 않은지 확인한다.
6. 표준 HAZOP와 크게 다른 판단이면 확인 필요로 표시한다.
7. 사고이력이나 표준 HAZOP 데이터가 부족한 경우 그 부족함을 근거에 남긴다.
