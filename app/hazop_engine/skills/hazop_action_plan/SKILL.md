# HAZOP 조치계획서 작성 절차

1. 위험도 9 이상인 위험성평가 Row만 대상으로 한다.
2. 현재안전조치에서 부족한 부분을 찾는다.
3. 개선권고사항을 구체적으로 작성한다.
4. 조치 후 빈도 후보와 근거를 작성한다.
5. 조치 후 강도 후보와 근거를 작성한다.
6. 조치 후 위험도는 시스템 코드가 다시 계산하므로 직접 확정하지 않는다.
7. 실제 적용 가능성 확인이 필요한 경우 비고에 남긴다.
