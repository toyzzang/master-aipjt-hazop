# 표준 HAZOP 참조 절차

1. 사용자가 입력한 표준공정위험성평가서 Link 또는 ID를 확인한다.
2. Node, 변수, Guideword, 물질이 유사한 항목을 찾는다.
3. 유사 항목의 원인, 결과, 현재 안전조치, 빈도, 강도를 참고한다.
4. 현재 설비와 다른 점은 그대로 복사하지 말고 확인 필요로 남긴다.
5. 실제 문서를 조회할 수 없으면 참조값은 기록하고 `표준 HAZOP 문서 확인 필요`를 남긴다.
