# HAZOP 조치계획서 작성 절차

1. 시스템이 선별한 위험도 9 이상 위험성평가 Row만 대상으로 한다.
2. 현재 안전조치가 원인을 예방하는지, 결과를 완화하는지 나누어 부족한 부분을 찾는다.
3. 개선권고사항을 실행 가능한 문장으로 구체적으로 작성한다.
4. 정보가 부족하면 MSDS 상세 조회, 사고이력 조회, 표준 HAZOP 조회 Tool을 사용한다.
5. 조치 후 빈도 후보와 근거를 작성한다.
6. 조치 후 강도 후보와 근거를 작성한다. 물질 자체의 위험성과 최대 사고 영향이 바뀌지 않으면 강도를 임의로 낮추지 않는다.
7. 조치 후 위험도는 시스템 코드가 다시 계산하므로 직접 확정하지 않는다.
8. 실제 적용 가능성 확인이 필요한 경우 비고에 남긴다.
